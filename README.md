# astrbot_bottle
云数据版本的漂流瓶

所有投放过的漂流瓶将会存储在云数据库,内容人工审核,不会担心出现有害的漂流瓶
漂流瓶主页地址:https://rnrqsj.top/api/a/home.php
调用API文档参考:https://rnrqsj.top/?action=doc&id=2
目前导入到astrbot有些问题,个人刚接触过开发插件,不太很会

以下是指令
普通用户指令	捡漂流瓶 / 捡瓶	捡一个漂流瓶	捡瓶
普通用户指令	投漂流瓶 文字内容（+ 图片）	投放漂流瓶	投漂流瓶 今天去看海啦～（附带图片）
普通用户指令	我的漂流瓶 / 漂流瓶统计	查询个人今日次数	我的漂流瓶
管理员指令	漂流瓶管理 查询 QQ 号	查询指定用户次数	漂流瓶管理 查询 123456789
管理员指令	漂流瓶管理 重置 QQ 号	重置指定用户次数	漂流瓶管理 重置 123456789
管理员指令	漂流瓶管理 全局统计	查看今日全服数据	漂流瓶管理 全局统计
